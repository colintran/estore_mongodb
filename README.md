# Estore in Expressjs
## Stack
- Express
- Mongodb, mongoose
- Model: MVC
## Pre-requisite
- Mongodb installed beforehand
  - A database is allocated for estore, its mongodb connection string needs to be set in variable **MONGO_URL** in [`app.js`](app.js#L8) file
## Installation
- `npm install`
- `node app.js`
